# [Di Hei's Blog](http://diheiblog.com) 

Powered by [Jekyll](http://jekyllrb.com/) on [GitHub Pages](http://pages.github.com), color theme inspired by [lanyon](https://github.com/poole/lanyon).

### Quick-start

##### Install chocolatey (on Windows) if you don't already have it
    iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))

##### Install ruby via chocolatey (on Windows) if you don't already have it
    choco install ruby -y

##### Clone the repository
    git clone https://github.com/dhei/dhei.github.io
    cd ./dhei.github.io/

##### Install Jekyll and dependencies
	gem install jekyll
    gem install jekyll-paginate

##### Run Jekyll to serve locally
	jekyll serve

	# => Now browse to http://localhost:4000/
